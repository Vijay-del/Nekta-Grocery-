💡 Tips & Rules
• Weight items (unit = Kg): Always set halfPrice. quarterPrice is optional but recommended.
• halfPrice must be strictly less than price. Otherwise the upload will reject the row.
• Images: Put images in the /images/ folder. Just write the filename in the column.
• Update mode: Match is done by product name (case-insensitive). Name must match exactly.
• Stock only mode: Only updates stock + outOfStock. Everything else stays the same.
• Max batch size: 500 products per upload. Split into multiple files if larger.
• Currency: All prices in ₹ (Indian Rupees). No symbols needed in the cell.
